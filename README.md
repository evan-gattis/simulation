Unity Ecosystem Simulation - OOP Spring 2026
Project members: Evan Gattis, Kevin Kauffman

Requirements:
- Unity 6 or later
- Unity AI Navigation package (com.unity.ai.navigation 2.0.12)

Patterns Implemented:
- Strategy: eat/target strategies
- Factory: creates wolfes/sheep
- Builder: Builds the ecosystem
- Observer: Lets wolves observe sheep

To run:
1. Open the project in Unity Hub
2. Open the scene in Assets/Scenes
3. Press Play

To run tests:
1. Open Window > General > Test Runner
2. Select EditMode or PlayMode tab
3. Click Run All
